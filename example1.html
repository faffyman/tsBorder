jQuery tsBorder Examples

tsBorder allows you to overlay a semi transparent border on block elements.
You can choose your preferred border colour, opacity and thickness in pixels.

Landscape Image gets a 10px Black border set to 0.3 (30%) opacity 

      $('#elementID').tsBorder({
                           thickness: 10,
                           color: '#000000',
                           opacity:0.3
                          });

Portrait Image gets a 15px Whiteborder set to 0.5 (50%) opacity 
      $('#elementID').tsBorder({
                           thickness: 15,
                           color: '#ffffff',
                           opacity:0.5
                          });
Extreme - example: Image gets a 78px Green border set to 0.8 (80%) opacity 
      $('#elementID').tsBorder({
                           thickness: 78,
                           color: '#00ff00',
                           opacity:0.8
                          });
Image Floated right. Floated elements sometimes go a bit funny with the standard implementation. As seen here.
However the issue is more related to margins that the floating. There are ways to manually correct this seen below. 
      $('#elementID').tsBorder({
                           thickness: 10,
                           color: '#ffffff',
                           opacity:0.5
                          });
If an element is floated or already has a border property, or for some reason you just want to shift the overlay border, you can use the offset properties. 
      $('#elementID').tsBorder({
                           thickness: 10,
                           color: '#000000',
                           opacity:0.3,
                           offsetLeft: 5,
                           offsetTop: 5
                          });
Image Floated right. This time we use the offsetLeft property to correct for the left margin of the floated element. 
      $('#elementID').tsBorder({
                           thickness: 10,
                           color: '#ffffff',
                           opacity:0.5,
                           offsetLeft:20
                          });
In this example the floated element doesn't use an offsetleft property.
That's because there is no left margin 

      $('#elementID').tsBorder({
                           thickness: 10,
                           color: '#ffffff',
                           opacity:0.7,
                          });
What happen if you add multiple borders? They stack rather than nest, that's what.
See that the first thicker red border is under the thinner green border, giving the effect of an inset. 
      $('#elementID').tsBorder({
                           thickness: 20,
                           color: '#ff0000',
                           opacity:0.6,
                          }).tsBorder({
                           thickness: 10,
                           color: '#00ff00',
                           opacity:0.5,
                          });

Notice how the image gets a 10px red border - but the outer div element gets a 3px black tsBorder layer.
The Green line is the css border property on the div element - this is always outside the element.

See how see how margins and padding effect the position of the borders.
You can use an extra option called offset to try to shift the borders.

As with any jQuery selector you can apply the effect on multiple elements with a single call 

          $('.thickBlack').tsBorder({
                           thickness: 15,
                           color: '#000000',
                           opacity:0.7,
                          });

          $('.thickBlack').tsBorder({
                           thickness: 4,
                           color: '#00ff00',
                           opacity:0.7,
                          });
